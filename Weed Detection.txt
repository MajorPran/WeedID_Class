https://www.kaggle.com/datasets/ravirajsinh45/crop-and-weed-detection-data-with-bounding-boxes/data  :- This data set contains only 2 distinct classes, Crop and Weed. The crop is believed to sesame plant while the weed is a generalised category. This dataset is created by an Indian so the weed data is probably from India as well.
